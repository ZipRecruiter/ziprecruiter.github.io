Using the Bookmarklets

These bookmarklets clean out some of the Mixpanel Segmentation stuff (namely the checkbox information on the graphs) to make the URLs much shorter.

To use, drag the links below to your bookmarks bar and click them while on a Mixpanel "Segmentation" page.

  1. This bookmarklet cleans the mixpanel segmentation URL and allows you to copy it from a JavaScript prompt:

    Get Clean Mixpanel URL

    Mainly for sharing shorter urls with others.

  2. Clean the mixpanel segmentation URL and go to the new URL:

    Go To Clean Mixpanel URL

    This bookmarklet is for fixing a situation where the graph shows you unwanted (old) values. Kind of an edge case.
